CSS Definition

CSS, or Cascading Style Sheets, is a language that is used in combination with HTML that customizes how HTML elements will appear. CSS can define styles and change the layout and design of a sheet.

Create a CSS file

You can create a css file by attaching .css extension

Attaching CSS to HTML

There are two ways of doing it.

  • Create a separate CSS file.
  • Directly into the head tag
1<head>
2  <link href="style.css" type="text/css" rel="stylesheet">
3</head>
1<head>
2 <style>
3  p{
4    font-size: 1000px;
5  }
6 </style>
7</head>

Styling an element

Multiple ways of doing it. Here are some examples.

  • Inline styles

CSS styles can be directly added to HTML elements by using the style attribute in the element’s opening tag. Each style declaration is ended with a semicolon. Styles added in this manner are known as inline styles.

1<h2 style="text-align: right;">Right Aligned</h2>
2 
3<p style="background-color: blue; font-size: 18px;">BG Blue, 18-point text</p>
  • Class and ID selectors

CSS classes can be reusable and applied to many elements. Class selectors are denoted with a period . followed by the class name. CSS ID selectors should be unique and used to style only a single element. ID selectors are denoted with a hash sign # followed by the id name.

1/* Selects all elements with class="image_small" */
2.image_small {
3}
4 
5/* Selects element with id="first-header" */
6#first-header {
7}

Selectors can be chained

1/* Selects all elements with class="image_small" and class="image_blue_section" */
2.image_small.image_blue_section {
3}

Selectors can be grouped

1/* Selects elements h1 and h2 and applies color style: blue */
2h1,h2 {
3 color:blue;
4}

Selectors examples

CSS Type Selectors

CSS type selectors are used to match all elements of a given type or tag name. Unlike for HTML syntax, we do not include the angle brackets when using type selectors for tag names. When using type selectors, elements are matched regardless of their nesting level in the HTML.

1/* Selects all <div> tags */
2div {
3}

CSS class selectors

The CSS class selector matches elements based on the contents of their class attribute. 

1/*For selecting elements having cell as the value of the class attribute, a . needs to be prepended.*/
2.cell {
3  color: #000;
4}

CSS ID selectors

The CSS ID selector matches elements based on the contents of their id attribute. 

1/*For selecting the element having cell as the value of the id attribute, a # needs to be prepended.*/
2#cell {
3  color: #000;
4}

CSS descendant selector

The CSS descendant selector combinator is used to match elements that are descended from another matched selector. They are denoted by a single space between each selector and the descended selector. All matching elements are selected regardless of the nesting level in the HTML.

1div p { }
2 
3section ol li { }

CSS Properties Syntax

The property name and value are separated by a colon, and the entire declaration must be terminated by a semi-colon. These declarations are called key-value pair.

1/* 
2CSS declaration format:
3property-name: value;
4*/
5 
6/* CSS declarations */
7text-align: center;
8color: purple;
9width: 100px;
